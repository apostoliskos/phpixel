# phpixel.gr

Εταιρικό site της phpixel: κατασκευή e-shop, ιστοσελίδες, SEO & AEO, email marketing και custom εφαρμογές.

Χτισμένο με [Astro 5](https://astro.build), φιλοξενείται στο Vercel (SSR + prerendered σελίδες).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Το deploy γίνεται αυτόματα από το Vercel σε κάθε push στο `main`.
